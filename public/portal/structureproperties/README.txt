How to run this demo:
1) Unzip
2) In this folder, run: python -m http.server 8000
3) Open: http://localhost:8000/index.html

To add a thumbnail for a module:
- Put thumb.png in modules/structureproperties/<module-id>/thumb.png

To make the Test video work:
- Put video.mp4 in modules/structureproperties/test/video.mp4
